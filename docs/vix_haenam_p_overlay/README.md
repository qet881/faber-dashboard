# 해남P + VIX 오버레이 백테스트 메모

## 기준

- 실행일: 2026-06-17
- 기간: 2000-01-03 ~ 2026-06-17
- 기본 엔진: 앱의 해남P 엔진
- VIX 신호: 한국 체결 시점의 룩어헤드를 줄이기 위해 전일 사용 가능한 VIX 종가 사용
- VIX 매수:
  - VIX 25~39: 하루 최대 +1%p
  - VIX 40 이상: 하루 최대 +10%p
  - VIX 80 이상: 즉시 목표까지
- 매수 대상: 코스피200 슬롯과 나스닥 슬롯 50:50 추가
- VIX 활성 중 비주식 잔여분: 현금
- 매도/복귀: 주식 회복 바스켓이 전고점을 회복하면 해남P 기본비중으로 복귀

## 해석

현재 정의한 VIX 오버레이는 해남P보다 좋아 보이지 않는다.

- CAGR이 해남P보다 낮다.
- MDD가 약 -9%대에서 -23%대로 커졌다.
- 월말 MDD도 약 -5.6%에서 -20%대로 커졌다.
- 전고점 회복 매도 조건이 너무 엄격해서 VIX 사이클이 오래 유지된다.

이 결과만 보면 VIX 오버레이를 실전 기본값으로 채택할 이유는 약하다. 다만 앱에 연구용 비교군으로 남겨두고, 매도 조건을 더 짧게 만든 변형을 별도 테스트할 가치는 있다.

후속 후보:

- 전고점 회복이 아니라 전고점 -5% 회복 시 전량 축소
- VIX 20 이하 안정화 + 200일선 회복 시 월별 축소
- VIX 상한을 70%가 아니라 50~60%로 제한
